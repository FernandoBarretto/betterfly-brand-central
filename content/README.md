# Betterfly Brand Central — Content Library

Drop brand files directly into this folder (or subfolders) and they will automatically appear in the Brand Central content library.

Supported file types:
- PDFs (.pdf)
- PowerPoint / Google Slides exports (.pptx, .ppt)
- Word / Google Docs exports (.docx, .doc)
- Images (.png, .jpg, .jpeg, .gif, .svg, .webp)
- Videos (.mp4, .mov, .webm)
- Excel / Google Sheets exports (.xlsx, .xls)

Naming tips — the file name is used to infer its category:
- Include "webinar" or "recording" → filed under Webinars
- Include "one-pager" or "onepager" → One-Pagers
- Include "case study" or "case-study" → Case Studies
- Include "competitive" or "competitor" → Competitive Analysis
- Include "research" or "report" → Research
- Include "blog" or "article" → Blog / Articles
- Include "template" → Templates
- Include "pitch" or "deck" → Presentations
- Anything else → Other

After dropping files here, restart the dev server (or redeploy) to see them in the app.
